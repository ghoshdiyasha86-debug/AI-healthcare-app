# 🏥 AI Smart Healthcare Assistant

## 📌 Problem
Many people cannot understand their symptoms or reports properly and delay visiting doctors.

## 💡 Solution
This app allows users to:
- Enter symptoms
- Upload medical reports
- Upload images
- Get possible condition analysis
- Get doctor recommendations

## ⚙️ Features
- Symptom-based analysis
- Image and report upload
- Doctor suggestion (type, budget, availability)
- Clean UI with animations

## 🛠 Tech Stack
- Python
- Streamlit
- CSS (Custom UI)
